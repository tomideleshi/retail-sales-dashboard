# Retail Sales Interactive Dashboard

## Project Overview
An end-to-end sales analytics solution built entirely in Google Sheets and Google Looker Studio, analyzing retail performance across salespeople, regions, and product categories.

## Tools Used
- Google Sheets (Advanced Excel functions)
- Google Looker Studio
- Advanced Excel: VLOOKUP, SUMIF, INDEX/MATCH, Data Validation

## Key Features
- Interactive dashboard with dynamic filters for Region and Date
- Bar charts, pie charts, and scorecards for quick business insights
- Advanced formulas including SUMIF salesperson lookup and VLOOKUP product price lookup
- Color coded data validation dropdowns to enforce data integrity

## Key Insights
- Total revenue analyzed: $61,000+
- Best performing region: West
- Top product category: Laptop
- Dashboard enables real-time filtering across 5 salespeople and 5 product lines
